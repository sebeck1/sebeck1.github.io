Verschiebung von Funktionen

Im Folgendem werden Sie sich verschiedene Möglichkeiten angucken, eine Funktion mithilfe von Parametern zu verändern.

Veränderung der Funktion durch f(x)+c

Arbeitsauftrag:

  1. Geben Sie verschiedene Funktion in das obige Applet ein, dass der Form f(x)+c entspricht.
  2. Geben Sie nun anstelle einer Zahl einen Buchstaben in das Applet ein und nutzen Sie den dann erscheinenen Regler, um sich deutlich zu machen, welchen Einfluss der Parameter auf die Funktion nimmt.
  3. Schreiben Sie anschließend ihre Beobachtungen in ihr Heft.
